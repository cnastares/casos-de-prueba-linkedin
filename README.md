# **inicio de sesión en LinkedIn**

**Descripción**

Yo como usuario registrado, quiero iniciar sesión en el sistema
de LinkedIn.

![Untitled](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/7b870b05-e764-40fd-89e7-2402eb8e2f6a/Untitled.png)

**Criterios de aceptación**

1. **Botón de iniciar sesión en web**
    
    Dado que un usuario registrado se encuentra en la pagina web principal de LinkedIn.
    
    Entonces se muestra un botón de iniciar sesión destacado arriba a la derecha, junto al botón de unirse ahora.
    
2. **Botón de iniciar sesión en Mobile**
    
    DADO que un usuario registrado se encuentra en la aplicación móvil
    ENTONCES se muestra iniciar sesión destacado en primer lugar, luego en la parte inferior se muestra los botones de iniciar sesión con Google, iniciar sesión con Apple y iniciar sesión con Facebook.
    
3. **Acceder a pantalla de iniciar sesión - Web**
    
    DADO un usuario registrado que se encuentra en la página principal de LinkedIn
    CUANDO hace click en el botón Iniciar sesión.
    ENTONCES se muestra la pantalla de login con los siguientes
    elementos en pantalla:
    Logo de LinkedIn
    Campo para Ingresar el email o teléfono. Leyenda: “email o teléfono”
    
    Campo para Ingresar la contraseña. Leyenda: “contraseña”
    
    Link de has olvido tu contraseña
    Botón azul “iniciar sesión”
    Botón para iniciar sesión con Google
    
    Botón para iniciar sesión con Apple
    Link de unirse ahora
    
4. **Acceder a pantalla de iniciar sesión - Mobile**
    
    DADO un usuario registrado desea iniciar sesión en la app de LinkedIn
    CUANDO hace click en la app de LinkedIn.
    ENTONCES se muestra la pantalla de login con los siguientes
    elementos en pantalla:
    Logo de LinkedIn
    Campo para Ingresar el email o teléfono. Leyenda: “email o teléfono”
    
    Campo para Ingresar la contraseña. Leyenda: “contraseña”
    
    Link de has olvido tu contraseña
    Botón azul “iniciar sesión”
    Botón para iniciar sesión con Google
    
    Botón para iniciar sesión con Apple
    Link de unirse ahora
